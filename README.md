# Vanilla-Plus
A mod package that improves vanilla Minecraft in performance and features

Current Version: 1.3.0

Supported versions: 1.20.X, 1.21.X

### Mods:

- 3D Skin Layer
- AppleSkin
- Architectury API
- Chat Heads
- Cloth Config API
- ESSENTIAL MOD
- Dynamic Crosshair
- FabricAPI
- Fabric Language Kotlin
- Freelook
- Indium
- Iris Shaders
- Lithium
- Mod Menu
- Roughly Enough Items (REI)
- Shulker Box Tooltip
- Sodium
- Xaero's Minimap (Fair-Play)
- Xaero's World Map
- YetAnotherConfigLib
- Zoomify
  
### Ressource packs:

- Faithful
- Default Dark Mode
- No Pumpkin Blur

### Shader:
- Complementary Reimagined

### Changes:
Things that have been changed

- Zoomify:
    - The zoom button has been set to **Z**
      
- Xaeros's Minimap:
    - The button for enlarging the minimap has been changed to **C**

## Info:
This modpack is not finished yet and new mods/ressourcepacks/shaders will be added.
